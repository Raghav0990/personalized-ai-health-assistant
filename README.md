#  Personalized AI Health Assistant

![App Screenshot](screenshot.png)

##  Overview
A multimodal AI-based system that predicts health risks using:
- Medical history
- Wearable data (heart rate, steps)
- Explainable AI techniques

##  Features
- ✔ Health Risk Prediction (HIGH / LOW)
- ✔ Explainable AI insights
- ✔ Interactive Streamlit dashboard
- ✔ Modular ML pipeline

##  Tech Stack
- Python
- Scikit-learn
- Streamlit
- NumPy / Pandas

pip install -r requirements.txt
streamlit run app.py
